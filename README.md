# pie_chart
A small Java, MVC project

## Description
A pie chart program ![pie chart image](doc/images/pie_chart.png)

## How to install (Mac)
- git clone https://github.com/torgammelgard/pie_chart.git
- cd pie_chart
- ./gradlew build
- java -jar ./build/libs/pie_chart.jar

### Intellij
To create Idea project :

- ./gradlew idea
