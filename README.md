## Description
A pie chart program

## How to install (Mac)
- git clone https://github.com/torgammelgard/pie-chart.git
- cd pie-chart
- ./gradlew build
- java -jar ./build/libs/pie-chart.jar

### Intellij
To create Idea project :

- ./gradlew idea
